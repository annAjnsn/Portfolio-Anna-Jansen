About

As mentioned in the previous page, this website serves as a digital portfolio of the maps and apps I made for a GIS lab course. The overarching theme for all features is land use and quality. The inspiration for this theme came from the UN’s SDG ( Sustainable Development Goal) number 15, which is about the protection and sustainable usage of land on earth. In the ‘Maps’ section you will find my work showcased.

My Maps

Below are all the maps and web apps I have created. Clicking on them will open up the application in a separate browser. From top to bottom, the first two maps show SDG indicator 15.3.1, which is about land degradation. The first map is about data finalized in 2015 and the second one about data from 2019. The next web app is based on a Mobile Application, asking for land use suitability estimations of a certain area (should you want to contribute to this data, you can do so by clicking here).The fourth feature is a web app and the online interactive version of the second map. The fifth and final map is an online map via Google Earth Engines that portrays the vegetation and land type cover of the southern main island of New Zealand. Within this map you can select the layer you would like to see, in which ‘false colour’ is the vegetation cover and ‘true colour’ a general topographic map. Feel free to browse through and explore them.
